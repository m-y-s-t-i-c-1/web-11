Bucle în JavaScript: for și while

🔁 Bucla for

Bucla for este folosită atunci când știm de câte ori vrem să repetăm o acțiune.

for (let i = 1; i <= 5; i++) {
  console.log("Numărul este: " + i);
}

Explicație:

  • let i = 1 – inițializarea (variabila începe de la 1)
  • i <= 5 – condiția (bucla continuă cât timp i este ≤ 5)
  • i++ – incrementarea (crește i cu 1 la fiecare pas)

🔄 Bucla while

while repetă un bloc de cod atâta timp cât o condiție este adevărată. Este utilă când nu știm exact câte repetări vor fi.

let i = 1;

while (i <= 5) {
  console.log("Valoarea este: " + i);
  i++;
}

Notă: Dacă nu actualizăm corect variabila (i++), bucla poate deveni infinită!

⚠️ Diferențe cheie între for și while

  • for: ideal când numărul de repetări este cunoscut
  • while: ideal când se repetă până se îndeplinește o condiție
  • Ambele pot realiza aceleași lucruri, dar alegerea depinde de claritatea și scopul codului
