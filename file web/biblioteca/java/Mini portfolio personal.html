Mini Portofoliu Personal cu HTML și CSS

În această lecție construim un site simplu de prezentare personală: un portofoliu cu informații despre tine și proiectele tale

Ce este un portofoliu?

Un portofoliu personal este un site unde îți prezinți:

  • numele și o scurtă descriere
  • proiectele la care ai lucrat
  • moduri de contact

Structura site-ului nostru

  1. Header cu nume + descriere
  2. Secțiune „Despre mine”
  3. Secțiune „Proiecte”
  4. Secțiune „Contact”
