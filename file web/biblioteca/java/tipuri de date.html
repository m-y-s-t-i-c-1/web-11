Tipuri de Date în JavaScript

📌 Tipul string

Un string este o secvență de caractere, adică text. Poate fi scris între ghilimele simple (' ') sau duble (" "), dar și cu backticks (` `) pentru template strings.

Exemple de utilizare:

let nume = "Ana";
let mesaj = 'Salut!';
let complet = `Bună, ${nume}!`;

console.log(typeof nume); // "string"

Notă: Template strings permit inserarea de variabile direct în text folosind ${...}.

📌 Tipul number

Tipul number reprezintă atât numere întregi cât și zecimale. Nu există tip separat pentru întregi (ca în alte limbaje).

JavaScript suportă operații matematice de bază, iar valorile pot fi pozitive, negative sau speciale (Infinity, NaN).

let varsta = 25;
let pret = 19.99;
let suma = varsta + pret;

console.log(typeof suma); // "number"

Notă: Dacă faci o operație invalidă, vei obține NaN (Not a Number).

📌 Tipul boolean

Un boolean poate avea doar două valori: true (adevărat) sau false (fals).

Este folosit frecvent în condiții (if/else), comparații și bucle.

let esteLogat = true;
let arePermisiune = false;

console.log(typeof esteLogat); // "boolean"

Notă: Poți obține valori booleene și din expresii logice:

let test = 5 > 3; // true
console.log(test);
