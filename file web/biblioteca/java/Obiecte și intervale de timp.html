🕒 Obiecte și Intervale de Timp în JavaScript

📅 Crearea obiectelor Date

// Data curentă
let acum = new Date();

// Data din milisecunde
let t0 = new Date(0);

// Data din șir
let dataText = new Date("2025-04-16T12:00:00");

// Data detaliată: an, lună (0-11), zi, oră, minute
let detaliat = new Date(2025, 3, 16, 12, 30);

console.log(acum);

🔍 Obținerea și setarea componentelor

let data = new Date();

console.log(data.getFullYear());
console.log(data.getMonth());
console.log(data.getDate());

// Setare
data.setFullYear(2030);
data.setMonth(0); // ianuarie

📆 Formatarea datelor

let data = new Date();

console.log(data.toLocaleDateString("ro-RO", {
  year: "numeric",
  month: "long",
  day: "numeric"
}));

console.log(data.toLocaleTimeString());

🧮 Calculul diferențelor între date

let inceput = new Date("2025-04-01");
let sfarsit = new Date("2025-04-16");

let diferenta = sfarsit - inceput; // în milisecunde
let zile = diferenta / (1000 * 60 * 60 * 24);
console.log("Zile între date:", zile);
