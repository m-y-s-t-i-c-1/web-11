Cum funcționează fișierele în JavaScript?

Fișierele JavaScript sunt folosite pentru a adăuga funcționalități interactive pe site-uri web. Aceste fișiere sunt de obicei separate de fișierele HTML și CSS pentru a permite o mai bună organizare și întreținere a codului.

Pentru a le folosi într-un document HTML, le legăm cu ajutorul tag-ului <script>:

<script src="script.js"></script>

Folosirea modulelor în JavaScript

Modulele JavaScript permit organizarea codului în fișiere separate. Fiecare fișier poate exporta funcții sau variabile pe care alte fișiere le pot importa.

Exemplu de modul:

// fișierul math.js
export function aduna(a, b) {
    return a + b;
}

// fișierul main.js
import { aduna } from './math.js';
console.log(aduna(5, 3));

Cu modulele, poți separa funcționalitățile într-o aplicație mare într-un mod clar și ușor de întreținut.

Structura unui proiect JavaScript

Este important să îți organizezi fișierele într-o structură logică. Iată un exemplu de structură pentru un proiect JavaScript:

/proiect
|-- /css
|-- /js
|   |-- app.js
|   |-- utils.js
|   |-- math.js
|-- index.html

Acesta este un mod eficient de a păstra fișierele organizate, astfel încât să poți lucra ușor cu ele pe măsură ce proiectul crește.

Folosirea LocalStorage și SessionStorage

LocalStorage și SessionStorage sunt două metode de stocare a datelor în browser. Acestea îți permit să salvezi informații pentru utilizatori, astfel încât datele să fie persistente între sesiuni.

Exemplu de utilizare a LocalStorage:

// Salvarea unui element în LocalStorage
localStorage.setItem('nume', 'Ion');

// Preluarea elementului din LocalStorage
let nume = localStorage.getItem('nume');
console.log(nume);  // Ion

LocalStorage stochează datele pe termen lung, în timp ce SessionStorage este folosit pentru a salva date doar pentru durata sesiunii curente.

Exemple de organizare în aplicații complexe

În aplicații mai mari, este important să ai o organizare bună a fișierelor și a componentelor. De exemplu, poți folosi framework-uri precum React sau Vue.js, care promovează o structură modulară a codului.

În React, de exemplu, fișierele pot fi organizate în funcție de componente, iar fiecare componentă poate fi un fișier separat:

/src
|-- /components
|   |-- Header.js
|   |-- Footer.js
|   |-- Content.js
|-- App.js
