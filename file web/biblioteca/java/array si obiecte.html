Array de Obiecte în JavaScript

Ce este un Array de Obiecte?

Un array de obiecte este un array care conține obiecte. Fiecare obiect poate avea mai multe proprietăți. Este util atunci când vrei să grupulezi mai multe elemente cu aceeași structură, dar cu date diferite.

Exemplu de Array de Obiecte

Următorul exemplu creează un array cu obiecte ce conțin informații despre utilizatori:

let users = [
  { name: "Ion", age: 30, job: "Programator" },
  { name: "Maria", age: 25, job: "Designer" },
  { name: "Alex", age: 35, job: "Manager" }
];

console.log(users[0].name); // Ion
console.log(users[1].age); // 25
console.log(users[2].job); // Manager

Manipularea Elementelor din Array

Poți accesa, modifica sau adăuga elemente în array folosind indexul sau prin metode JavaScript. Iată un exemplu:

Modificarea vârstei unui utilizator:

users[0].age = 31;
console.log(users[0].age); // 31

Filtrarea și Transformarea Array-urilor

Poți utiliza metode precum filter() și map() pentru a manipula array-urile de obiecte. Exemplu:

Filtrarea utilizatorilor care au vârsta mai mare de 30:

let olderThan30 = users.filter(user => user.age > 30);
console.log(olderThan30);

Crearea unui nou array doar cu numele utilizatorilor:

let userNames = users.map(user => user.name);
console.log(userNames); // ["Ion", "Maria", "Alex"]

Considerații Finale

Un array de obiecte este o structură foarte utilă pentru gestionarea datelor structurate în aplicațiile JavaScript. Acesta poate fi folosit în mod eficient pentru a manipula datele din pagini dinamice, aplicații de backend sau pentru a lucra cu API-uri.
