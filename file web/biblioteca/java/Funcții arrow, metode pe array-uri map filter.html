Funcții Arrow și Metode pe Array-uri: map și filter

➡️ Funcții Arrow

Sintaxă:

(a, b) => a + b
// echivalent cu
function(a, b) {
  return a + b;
}

Avantaje: sintaxă mai scurtă, this legat lexical.

🧭 Metoda map()

Scop: Creează un nou array aplicând o funcție fiecărui element.

const numere = [1, 2, 3];
const dublate = numere.map(n => n * 2);
console.log(dublate); // [2, 4, 6]

Exemplu cu obiecte:

const persoane = [
  { nume: "Ana", varsta: 25 },
  { nume: "Ion", varsta: 30 }
];
const numePersoane = persoane.map(p => p.nume);
console.log(numePersoane); // ["Ana", "Ion"]

🔍 Metoda filter()

Scop: Creează un nou array doar cu elementele care trec un test.

const numere = [1, 2, 3, 4, 5];
const pare = numere.filter(n => n % 2 === 0);
console.log(pare); // [2, 4]

Exemplu cu obiecte:

const utilizatori = [
  { nume: "Ana", activ: true },
  { nume: "Ion", activ: false }
];
const activi = utilizatori.filter(u => u.activ);
console.log(activi); // [{ nume: "Ana", activ: true }]

🔗 Combinarea filter() + map()

const numere = [1, 6, 3, 8];
const rezultat = numere.filter(n => n > 5).map(n => n * 2);
console.log(rezultat); // [12, 16]

📏 Bune practici și performanță

console.time("map+filter");
const rez = numere.filter(n => n > 2).map(n => n * 3);
console.timeEnd("map+filter");

Recomandare: Folosește map() și filter() pentru claritate și concizie. Pentru performanță critică, buclele tradiționale pot fi mai rapide.
