🎯 Evenimente JavaScript – Esențialul

🔹 Ce sunt evenimentele?

Evenimentele sunt acțiuni declanșate de utilizator sau browser (click, tastare, scroll). Sunt cheia interactivității în pagini web.

document.getElementById("btn").addEventListener("click", () => {
  alert("Buton apăsat!");
});

🔸 Tipuri Comune

  • Mouse: click, mouseover, mouseout
  • Tastatură: keydown, keyup
  • Formulare: submit, focus, change
  • Pagină: load, resize, scroll
  • Tactile (mobil): touchstart, touchend

🔧 Cum adaugi evenimente

Recomandat:

element.addEventListener("event", functie);

Evită: HTML inline onclick="functie()"

Limitat: element.onclick = functie

🧩 Obiectul event

  • target – elementul declanșator
  • preventDefault() – oprește comportamentul implicit
  • stopPropagation() – oprește propagarea

🔁 Bubbling & Capturing

Bubbling: evenimentul urcă spre părinte

Capturing: evenimentul coboară de la root

Folosește useCapture în addEventListener() pentru control

📌 Delegarea Evenimentelor

document.querySelector("ul").addEventListener("click", (e) => {
  if (e.target.tagName === "LI") {
    console.log(e.target.textContent);
  }
});

Ideal pentru elemente adăugate dinamic.

🎉 Evenimente Personalizate

const ev = new CustomEvent("modificare", { detail: { stare: "nouă" } });
element.dispatchEvent(ev);

🚀 Avansat: Debounce & Throttle

let t;
window.addEventListener("resize", () => {
  clearTimeout(t);
  t = setTimeout(() => console.log("Redimensionare completă"), 300);
});

✅ Recomandări

  • Folosește addEventListener (nu onclick în HTML)
  • Delegă evenimente pentru elemente dinamice
  • Folosește debounce/throttle pentru performanță
  • Utilizează funcții săgeată pentru this
  • Comentează clar funcțiile de eveniment
