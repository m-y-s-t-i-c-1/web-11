Operatori și Condiții în JavaScript

🔢 Operatori de comparație

Operatorii de comparație sunt folosiți pentru a compara două valori. Rezultatul este întotdeauna un boolean: true sau false.

  • == — egal (compară valoarea, dar nu și tipul)
  • === — strict egal (compară și valoarea și tipul)
  • != — diferit
  • !== — strict diferit
  • >, <, >=, <= — mai mare, mai mic, etc.
let a = 5;
let b = "5";

console.log(a == b);  // true (doar valoarea)
console.log(a === b); // false (valoarea e aceeași, dar tipul diferit)

✅ Instrucțiunea if / else

if permite executarea unui bloc de cod doar dacă o condiție este adevărată. else este folosit pentru alternativa falsă.

let ora = 10;

if (ora < 12) {
  console.log("Bună dimineața!");
} else {
  console.log("Bună ziua!");
}

Poți adăuga mai multe ramuri cu else if:

let nota = 7;

if (nota >= 9) {
  console.log("Excelent");
} else if (nota >= 5) {
  console.log("Promovat");
} else {
  console.log("Respins");
}

🎯 Instrucțiunea switch

switch este folosit pentru a compara o valoare cu mai multe cazuri posibile. Este mai clar decât multe if / else consecutive.

let zi = "luni";

switch (zi) {
  case "luni":
    console.log("Începe săptămâna.");
    break;
  case "vineri":
    console.log("Sfârșit de săptămână aproape!");
    break;
  case "sâmbătă":
  case "duminică":
    console.log("Weekend!");
    break;
  default:
    console.log("Zi necunoscută.");
}

Notă: break oprește execuția după caz. Fără el, codul continuă cu următorul caz (fall-through).
