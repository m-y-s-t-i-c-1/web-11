localStorage în JavaScript

În JavaScript, localStorage este o metodă de stocare a datelor care rămân persistente între sesiunile de navigare, chiar și după închiderea și redeschiderea browser-ului.

Ce este localStorage?

localStorage permite stocarea datelor sub formă de perechi cheie-valoare. Aceste date rămân salvate în memoria locală a browserului și nu sunt șterse automat.

Metodele principale:

  • setItem(key, value): Stochează o valoare.
  • getItem(key): Recuperează valoarea.
  • removeItem(key): Șterge o valoare.
  • clear(): Șterge toate valorile.
  • key(index): Returnează cheia de la o poziție dată.

Exemplu simplu:

const name = "Ana";
localStorage.setItem("userName", name);

const retrievedName = localStorage.getItem("userName");
console.log("Numele salvat este:", retrievedName);

Ștergere:

localStorage.removeItem("userName");

Ștergere completă:

localStorage.clear();

Valori non-string:

Folosește JSON.stringify() pentru a salva obiecte și JSON.parse() pentru a le citi:

const user = { nume: "Ana", varsta: 22 };
localStorage.setItem("utilizator", JSON.stringify(user));

const dinStorage = JSON.parse(localStorage.getItem("utilizator"));
console.log(dinStorage.nume);
