🧠 Manipularea DOM-ului – Ghid Interactiv

🔍 Ce este DOM-ul?

DOM-ul (Document Object Model) este o reprezentare internă a paginii HTML în memorie. Fiecare element devine un „nod” pe care JavaScript îl poate accesa și modifica.

document.getElementById("exemplu")

Aceasta este o propoziție care va fi schimbată.

🔎 Selectarea Elementelor

Putem selecta elemente în mai multe moduri:

  • getElementById("id") – selectează un element cu ID-ul dat.
  • querySelector(".clasa") – selectează primul element ce corespunde selectorului CSS.
  • querySelectorAll(".clasa") – selectează toate elementele.

  • createElement() pentru a crea un element
