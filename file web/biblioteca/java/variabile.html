Variabile în JavaScript: let vs const

Ce este let?

let este folosit pentru a declara variabile care pot fi reasignate.

let numar = 5;
numar = 10;
console.log(numar); // 10

Este bloc-scoped, adică este vizibil doar în blocul în care este definit (ex: bucle, funcții).

Ce este const?

const este folosit pentru a declara constante – variabile care nu pot fi reasignate după ce sunt definite.

const pi = 3.14;
pi = 3.1415; // Eroare!

La fel ca let, este bloc-scoped.

⚠️ Dacă const este folosit cu obiecte sau array-uri, valorile interne pot fi modificate:

const persoana = { nume: "Ana" };
persoana.nume = "Ioana"; // Este permis
console.log(persoana);

Diferențe principale

  • let permite reasignare, const nu.
  • Ambele sunt bloc-scoped.
  • Evită folosirea var în favoarea let sau const.

Recomandare bună de practică

Folosește const implicit. Folosește let doar dacă știi că vei reasigna valoarea.
