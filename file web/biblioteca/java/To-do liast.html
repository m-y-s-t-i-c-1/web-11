Funcții și Array-uri în JavaScript

📦 Ce este o funcție?

O funcție este un bloc de cod care poate fi reutilizat pentru a executa o anumită acțiune. Poate primi parametri și poate returna o valoare.

function salut(nume) {
  return "Salut, " + nume + "!";
}

📚 Ce este un array?

Un array este o structură de date care poate conține mai multe valori într-o singură variabilă, accesibile prin index (începând de la 0).

let fructe = ["măr", "banană", "cireșă"];
console.log(fructe[0]); // măr
