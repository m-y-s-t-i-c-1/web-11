🚀 Asincron, Fetch & Async/Await în JavaScript

📦 Ce este fetch?

Metodă modernă de a face cereri HTTP. Returnează o promisiune.

🔧 Exemplu cu fetch și .then():

fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Eroare:", error));

✨ Async/Await

Scrii cod asincron ca și cum ar fi sincron. Ușor de citit și de gestionat erorile.

🔁 Exemplu cu async/await:

async function incarcaDate() {
  try {
    const raspuns = await fetch("https://jsonplaceholder.typicode.com/posts/1");
    const date = await raspuns.json();
    console.log(date);
  } catch (eroare) {
    console.error("Eroare la fetch:", eroare);
  }
}
