🔀 Instrucțiuni simple în C++

🔹 Ce sunt instructiuni simple?

Instrucțiunile simple permit programului să aleagă între mai multe căi de execuție, în funcție de condiții logice.

🔹 Instrucțiunea if

int x = 10;
if (x > 5) {
    cout << "x este mai mare decât 5" << endl;
}

Explicație: Dacă expresia x > 5 este adevărată, codul din acolade se execută.

🔹 if...else

int x = 3;
if (x >= 5) {
    cout << "x este cel puțin 5" << endl;
} else {
    cout << "x este mai mic decât 5" << endl;
}

🔹 if...else if...else

int nota = 8;
if (nota >= 9) {
    cout << "Excelent" << endl;
} else if (nota >= 7) {
    cout << "Bine" << endl;
} else {
    cout << "Necesită îmbunătățiri" << endl;
}

🔹 Instrucțiunea switch

int zi = 3;
switch (zi) {
    case 1:
        cout << "Luni"; break;
    case 2:
        cout << "Marți"; break;
    case 3:
        cout << "Miercuri"; break;
    default:
        cout << "Zi necunoscută";
}

Important: switch funcționează doar cu tipuri întregi sau char. Fără break, execuția continuă în jos.

🔹 Exercițiu propus

Scrie un program care citește un număr și afișează dacă este pozitiv, negativ sau zero.

#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Introdu un număr: ";
    cin >> n;

    if (n > 0)
        cout << "Numărul este pozitiv";
    else if (n < 0)
        cout << "Numărul este negativ";
    else
        cout << "Numărul este zero";

    return 0;
}
