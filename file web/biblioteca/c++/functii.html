🧩 Funcții în C++

🔹 Ce este o funcție?

O funcție este un bloc de cod care îndeplinește o anumită sarcină și poate fi apelat oricând avem nevoie.

🔹 Structura unei funcții

tip_return nume(parametri) {
    // cod
    return valoare;
}

Exemplu simplu:

int patrat(int x) {
    return x * x;
}

🔹 Apelul unei funcții

int r = patrat(5);
cout << "Patratul lui 5 este: " << r;

🔹 Funcții fără return

void salut() {
    cout << "Salutare!";
}

void indică faptul că funcția nu returnează nimic.

🔹 Funcții cu mai mulți parametri

int suma(int a, int b) {
    return a + b;
}

Apel: suma(3, 7) → returnează 10

🔹 Exercițiu

Scrie o funcție care verifică dacă un număr este prim:

bool estePrim(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++)
        if (n % i == 0)
            return false;
    return true;
}

Apel în main():

int n;
cin >> n;
if (estePrim(n))
    cout << "Este prim";
else
    cout << "Nu este prim";
