Tablouri Unidimensionale în C++

Ce este un tablou unidimensional?

Un tablou unidimensional (sau vector) este o colecție de elemente de același tip de date, stocate în memorie într-o succesiune continuă și accesate printr-un nume comun și un index.

  • Toate elementele sunt de același tip (int, float, char, etc.)
  • Elementele sunt stocate în locații consecutive de memorie
  • Accesul la elemente se face prin indici (poziții)
  • Primul element are indexul 0

Declararea tablourilor

Sintaxa pentru declararea unui tablou unidimensional:

tip_date nume_tablou[dimensiune];

Exemple practice:

// Declarare tablou de 5 întregi
int numere[5];

// Declarare cu inițializare
float temperaturi[4] = {36.5, 37.2, 36.8, 37.1};

// Declarare fără specificarea dimensiunii (determinată automat)
char vocale[] = {'a', 'e', 'i', 'o', 'u'};

Accesarea elementelor

Elementele unui tablou se accesează folosind operatorul de indexare []:

int vector[3] = {10, 20, 30};

// Accesare elemente
int primul = vector[0];  // 10
int al_doilea = vector[1]; // 20
vector[2] = 99;  // Modificare element

Important: C++ nu verifică limitele tabloului. Accesarea în afara limitelor poate cauza comportament nedefinit!

Parcurgerea tablourilor

Tablourile se parcurg de obicei cu bucle for:

int v[5] = {1, 2, 3, 4, 5};

// Parcurgere normală
for (int i = 0; i < 5; i++) {
    cout << v[i] << " ";
}

// Parcurgere cu range-based for (C++11)
for (int numar : v) {
    cout << numar << " ";
}

Dimensiunea tablourilor

Dimensiunea unui tablou (în bytes) se obține cu sizeof:

int arr[] = {1, 2, 3, 4, 5};
int dimensiune = sizeof(arr) / sizeof(arr[0]); // 5

Începând cu C++17, poți folosi std::size() din header-ul <iterator>:

#include <iterator>
int lungime = std::size(arr); // 5

Exemplu complet

#include <iostream>
using namespace std;

int main() {
    // Declarare și inițializare
    double note[5] = {9.5, 8.0, 7.5, 10.0, 6.5};
    
    // Calcul medie
    double suma = 0;
    for (int i = 0; i < 5; i++) {
        suma += note[i];
    }
    double media = suma / 5;
    
    cout << "Media notelor: " << media << endl;
    
    return 0;
}
