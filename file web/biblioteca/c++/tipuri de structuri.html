📦 Structuri de date în C++

🔹 1. Array (Tablouri statice)

Stochează o colecție de elemente de același tip cu dimensiune fixă.

int a[100]; // array de 100 de elemente
a[0] = 5;
cout << a[0];

🔹 2. Vector (din STL - <vector>)

Vectorii sunt tablouri dinamice care își pot schimba dimensiunea.

#include <vector>
using namespace std;

vector<int> v;
v.push_back(10);
v.push_back(20);
cout << v[0];

✅ Se poate adăuga, șterge sau redimensiona automat.

🔹 3. Structuri (tipuri de date personalizate)

Structurile grupează mai multe câmpuri de tipuri diferite.

struct Elev {
    string nume;
    int varsta;
    float media;
};

Elev e;
e.nume = "Ana";
e.varsta = 17;
e.media = 9.5;
cout << e.nume << " " << e.media;

🔹 Vectori de structuri

vector<Elev> clasa;

Elev e1 = {"Mihai", 16, 8.90};
Elev e2 = {"Ioana", 17, 9.60};

clasa.push_back(e1);
clasa.push_back(e2);

for (auto elev : clasa)
    cout << elev.nume << " " << elev.media << endl;

🔹 Concluzie

  • array – fix, rapid, simplu.
  • vector – flexibil, preferat în majoritatea cazurilor.
  • struct – creează tipuri de date compuse (ex: pentru elevi, produse etc).
