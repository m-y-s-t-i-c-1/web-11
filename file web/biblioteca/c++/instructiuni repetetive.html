🔁 Structuri repetitive în C++

🔹 Ce sunt structurile repetitive?

Structurile repetitive (buclă/loop) permit executarea repetată a unui bloc de cod cât timp o condiție este adevărată.

🔹 Bucla for

for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// Output: 1 2 3 4 5

Etape: inițializare → verificare condiție → executare → incrementare

🔹 Bucla while

int i = 1;
while (i <= 5) {
    cout << i << " ";
    i++;
}

Verifică condiția înainte de fiecare execuție.

🔹 Bucla do...while

int i = 1;
do {
    cout << i << " ";
    i++;
} while (i <= 5);

Execută cel puțin o dată, chiar dacă condiția e falsă de la început.

🔹 Exercițiu 1

Scrie un program care afișează toate numerele de la 1 la n (n citit de la tastatură):

#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "n = ";
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cout << i << " ";
    }
    return 0;
}

🔹 Exercițiu 2

Calculează suma numerelor de la 1 la n:

int suma = 0, i = 1;
while (i <= n) {
    suma += i;
    i++;
}
cout << "Suma este: " << suma;
