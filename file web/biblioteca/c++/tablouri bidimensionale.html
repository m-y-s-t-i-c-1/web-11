🔢 Matrici în C++

🔹 Ce este o matrice?

O matrice (tablou bidimensional) este o structură de date cu 2 dimensiuni: linii și coloane.

🔹 Declararea unei matrice

int a[100][100]; // matrice cu maxim 100 linii și 100 coloane

Elementele sunt accesate astfel: a[i][j]

🔹 Citirea și afișarea unei matrici

int m, n, a[100][100];
cin >> m >> n;
for (int i = 0; i < m; i++)
    for (int j = 0; j < n; j++)
        cin >> a[i][j];

for (int i = 0; i < m; i++) {
    for (int j = 0; j < n; j++)
        cout << a[i][j] << " ";
    cout << endl;
}

🔹 Suma elementelor din matrice

int suma = 0;
for (int i = 0; i < m; i++)
    for (int j = 0; j < n; j++)
        suma += a[i][j];
cout << "Suma elementelor: " << suma;

🔹 Elemente de pe diagonala principală (doar dacă matricea este pătratică)

for (int i = 0; i < n; i++)
    cout << a[i][i] << " ";

🔹 Exercițiu

Scrie un program care citește o matrice pătratică și determină elementul maxim de pe diagonala secundară.

#include <iostream>
using namespace std;

int main() {
    int n, a[100][100];
    cin >> n;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            cin >> a[i][j];

    int maxim = a[0][n-1];
    for (int i = 1; i < n; i++) {
        if (a[i][n-1-i] > maxim)
            maxim = a[i][n-1-i];
    }
    cout << "Maximul de pe diagonala secundara este: " << maxim;
    return 0;
}
