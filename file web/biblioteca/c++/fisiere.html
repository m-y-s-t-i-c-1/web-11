📁 Fișiere în C++

🔹 Bibliotecă necesară

Pentru lucrul cu fișiere, se folosește biblioteca <fstream>.

#include <fstream>

🔹 Citire din fișier

Presupunem că avem un fișier date.in cu numere.

#include <fstream>
#include <iostream>
using namespace std;

int main() {
    ifstream fin("date.in"); // deschidere fișier pentru citire
    int x;
    while (fin >> x) {
        cout << x << " ";
    }
    fin.close(); // închidere fișier
    return 0;
}

🔹 Scriere în fișier

#include <fstream>
using namespace std;

int main() {
    ofstream fout("date.out"); // deschidere fișier pentru scriere
    for (int i = 1; i <= 5; i++)
        fout << i * i << " ";
    fout.close(); // închidere fișier
    return 0;
}

🔹 Exercițiu practic

Fișierul input.txt conține n numere. Afișează suma lor în output.txt.

#include <fstream>
using namespace std;

int main() {
    ifstream in("input.txt");
    ofstream out("output.txt");

    int x, suma = 0;
    while (in >> x)
        suma += x;

    out << "Suma este: " << suma;

    in.close();
    out.close();
    return 0;
}

🔹 Observații

  • Fișierul trebuie să se afle în același director cu programul (sau să-i specifici calea completă).
  • Poți verifica dacă fișierul s-a deschis corect: if (!fin)
