Formulare în HTML

Eticheta <form> este folosită pentru a crea formulare în HTML. Acestea permit utilizatorilor să trimite informații către server.

Structura unui formular

<form action="procesare.php" method="post">
  <label for="nume">Nume:</label>
  <input type="text" id="nume" name="nume">
  <label for="mesaj">Mesaj:</label>
  <textarea id="mesaj" name="mesaj"></textarea>
  <button type="submit">Trimite</button>
</form>

Exemplu de formular

Explicație:

  • <form> definește formularul, iar atributul action specifică pagina care va procesa datele.
  • <label> este folosit pentru a eticheta câmpurile de introducere.
  • <input> este folosit pentru câmpurile de text și alte tipuri de date.
  • <textarea> este folosit pentru câmpuri de text mai lungi.
  • <button> este folosit pentru a trimite formularul.
