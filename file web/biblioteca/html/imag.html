Imagini în HTML

Pentru a afișa o imagine într-o pagină HTML, folosim eticheta <img>, care este o etichetă de tip „self-closing” (nu are închidere).

Structura unei imagini

<img src="imagine.jpg" alt="Descriere imagine">
  • src – calea către fișierul imaginii
  • alt – text alternativ (afișat dacă imaginea nu poate fi încărcată)

Exemplu cu imagine externă

<img src="https://via.placeholder.com/300" alt="Imagine exemplu">
