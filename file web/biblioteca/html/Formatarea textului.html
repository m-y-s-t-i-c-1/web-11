Formatarea Textului în HTML

HTML oferă mai multe etichete pentru a da stil și accent textului. Acestea sunt utile pentru evidențierea informațiilor importante.

Etichete uzuale

  • <strong> – text îngroșat (important)
  • <em> – text înclinat (accentuat)
  • <u> – text subliniat
  • <br> – linie nouă
  • <hr> – linie orizontală

Exemplu de utilizare

<p>Acesta este un <strong>text important</strong> și <em>text accentuat</em>.</p>
<p>Text cu <u>subliniere</u>. <br>Linie nouă aici.</p>
<hr>
<p>Text de după linia orizontală.</p>

Rezultatul afișat

Acesta este un text important și text accentuat.

Text cu subliniere.
Linie nouă aici.

Text de după linia orizontală.
