Liste în HTML

Listele sunt folosite pentru a organiza informația în mod clar. Există două tipuri principale de liste:

  • Listă neordonată – cu marcatori (bullets), folosind <ul>
  • Listă ordonată – cu numere, folosind <ol>

Exemplu de listă neordonată

<ul>
  <li>Măr</li>
  <li>Pere</li>
  <li>Cireșe</li>
</ul>

Rezultat:

  • Măr
  • Pere
  • Cireșe

Exemplu de listă ordonată

<ol>
  <li>Deschide editorul HTML</li>
  <li>Scrie codul</li>
  <li>Salvează fișierul</li>
</ol>

Rezultat:

  1. Deschide editorul HTML
  2. Scrie codul
  3. Salvează fișierul
