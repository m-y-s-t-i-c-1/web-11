Introducere în HTML

HTML (HyperText Markup Language) este limbajul standard pentru crearea paginilor web. Cu ajutorul HTML-ului, putem structura conținutul unei pagini web: titluri, paragrafe, imagini, linkuri și multe altele.

Structura de bază a unui document HTML

Un fișier HTML începe cu o structură de bază, ca în exemplul de mai jos:

<!DOCTYPE html>
<html>
  <head>
    <title>Titlul paginii</title>
  </head>
  <body>
    <h1>Salut, lume!</h1>
    <p>Aceasta este prima mea pagină HTML.</p>
  </body>
</html>

Explicație pe scurt

  • <!DOCTYPE html> – declară tipul documentului.
  • <html> – elementul principal care conține tot conținutul paginii.
  • <head> – secțiunea unde punem titlul și informații meta despre pagină.
  • <title> – textul care apare în titlul filei browserului.
  • <body> – conținutul vizibil al paginii: texte, imagini, butoane etc.

Ce urmează?

Acum că ai văzut cum arată o pagină HTML de bază, în următoarea lecție vei învăța despre titluri și paragrafe – elemente esențiale pentru structurarea conținutului.
