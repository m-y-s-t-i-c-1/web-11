Elemente div și span în HTML

Elementele <div> și <span> sunt utilizate în HTML pentru a grupa și stiliza conținutul, dar au scopuri diferite.

Elementul <div>

Elementul <div> este un container bloc care este utilizat pentru a grupa elemente mari de conținut, cum ar fi secțiuni, paragrafe, imagini sau alte grupuri de elemente. Acesta este utilizat mai ales pentru a structura pagina și a aplica stiluri CSS.

Exemplu de utilizare a <div>

<div class="secțiune">
  <h2>Secțiunea 1</h2>
  <p>Acesta este un paragraf în secțiunea 1.</p>
  </div>

Elementul <span>

Elementul <span> este un container inline care este folosit pentru a grupa elemente mici de conținut, de obicei pentru a aplica stiluri CSS asupra unor porțiuni de text dintr-un paragraf sau alt element inline.

Exemplu de utilizare a <span>

<p>Acesta este un text cu un cuvânt important: <span style="color:red">important</span>.</p>

Diferențele dintre <div> și <span>

  • <div>: Este un element de tip bloc, ceea ce înseamnă că ocupă întreaga lățime disponibilă și este folosit pentru gruparea elementelor mai mari.
  • <span>: Este un element inline, ceea ce înseamnă că se aliniază pe aceeași linie cu celelalte elemente și este folosit pentru a stiliza părți mici de conținut.

Rezumat

Elementele <div> și <span> sunt esențiale pentru structurarea și stilizarea conținutului într-o pagină HTML. <div> este utilizat pentru gruparea secțiunilor mai mari de conținut, în timp ce <span> este folosit pentru a aplica stiluri pe porțiuni mai mici de text sau alte elemente inline.
