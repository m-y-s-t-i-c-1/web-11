HTML - Linkuri (Hiperlinkuri)

Ce este un hiperlink?

Un hiperlink (sau simplu link) este un element care permite navigarea între pagini web. În HTML, linkurile sunt create cu elementul <a> (anchor).

Sintaxa de bază

<a href="URL">Text vizibil</a>

Unde href specifică adresa de destinație, iar conținutul dintre tag-uri este textul afișat.

Tipuri de linkuri

1. Linkuri absolute

<a href="https://www.example.com/pagina.html">Link absolut</a>

Conțin întreaga adresă URL.

2. Linkuri relative

<a href="pagina.html">Link relativ</a>

Se referă la resurse din același site.

3. Linkuri pe aceeași pagină

<a href="#sectiune">Salt la secțiune</a>
...
<h2 id="sectiune">Secțiunea țintă</h2>

Atribute importante

  • target - determină unde se deschide linkul: 
    <a href="https://example.com" target="_blank">Deschide în tab nou</a>
  • title - adaugă un tooltip: 
    <a href="https://example.com" title="Mergi la Example">Example</a>
  • rel - specifică relația dintre documente: 
    <a href="https://example.com" rel="nofollow">Example</a>

Exercițiu practic

Creează un link care:

  1. Deschide https://www.google.com într-un tab nou
  2. Are textul "Căutare Google"
  3. Afișează un tooltip "Deschide Google"
Soluție 
<a href="https://www.google.com" target="_blank" title="Deschide Google">Căutare Google</a>
