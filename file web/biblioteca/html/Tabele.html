Tabele în HTML

Etichetele pentru crearea tabelelor în HTML sunt <table>, <tr> (pentru rânduri), <th> (pentru celule de antet) și <td> (pentru celulele de date).

Structura unui tabel

<table>
  <tr>
    <th>Antet 1</th>
    <th>Antet 2</th>
  </tr>
  <tr>
    <td>Date 1</td>
    <td>Date 2</td>
  </tr>
</table>

Exemplu de tabel

Antet 1 Antet 2
Data 1 Data 2
Data 3 Data 4

Explicație:

  • <table> creează tabelul.
  • <tr> creează un rând.
  • <th> creează celulele de antet.
  • <td> creează celulele de date.
