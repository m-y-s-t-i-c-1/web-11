CSS - Pagina 2: Selectori

Selectori de bază

  • element – selectează toate elementele de acel tip (ex: p, h1)
  • .clasa – selectează toate elementele cu acea clasă
  • #id – selectează un element unic cu acel ID

Selectori combinați

  • div p – selectează toate <p> din interiorul unui <div>
  • ul > li – selectează elementele <li> direct în <ul> (nu și pe cele din subliste)
  • a:hover – stilizează linkul când treci cu mouse-ul peste el
