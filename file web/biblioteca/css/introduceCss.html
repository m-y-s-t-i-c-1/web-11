CSS - Pagina 1: Introducere

Ce este CSS?

CSS (Cascading Style Sheets) este un limbaj folosit pentru a stiliza paginile web. Cu ajutorul CSS, putem controla culorile, fonturile, marginile, dimensiunile și aspectul general al elementelor HTML.

Moduri de aplicare a CSS

  • Inline CSS: Direct în element HTML, prin atributul style.
  • CSS intern: Într-un bloc <style> în interiorul paginii HTML.
  • CSS extern: Într-un fișier separat (ex: stil.css) legat cu <link> în <head>.
