CSS - Pagina 6: Flexbox

Flex Direction

Proprietatea flex-direction determină direcția elementelor flexibile într-un container.

  • row - aliniere pe orizontală (implicit)
  • column - aliniere pe verticală
  • row-reverse - orizontal invers
  • column-reverse - vertical invers

Flex Wrap

Proprietatea flex-wrap controlează dacă elementele flexibile trec pe un nou rând sau coloană atunci când nu mai încap într-un container.

  • nowrap - implicit, toate elementele pe un singur rând
  • wrap - elementele vor "înfășura" pe următorul rând/coloană
  • wrap-reverse - wrap invers
