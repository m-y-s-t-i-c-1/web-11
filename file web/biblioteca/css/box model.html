CSS - Pagina 4: Box Model

Box Model este modul în care browserul calculează dimensiunile unui element HTML. Acesta include:

  • Content – conținutul efectiv (text, imagine etc.)
  • Padding – spațiu interior între content și border
  • Border – marginea vizibilă a elementului
  • Margin – spațiul exterior față de alte elemente
Aceasta este o cutie stilizată care evidențiază modelul box.

Exemplu explicativ

Content
+ Padding: 15px
+ Border: 2px dashed
+ Margin: 15px

Exercițiu

Creează un div cu:

  • Fundal gri deschis
  • Border solid de 3px albastru
  • Padding de 10px
  • Margin de 20px
  • Lățime de 250px
