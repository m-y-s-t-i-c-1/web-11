CSS - Pagina 5: Layout - Display, Position, Float

1. Proprietatea display

Determină cum este afișat un element:

  • block – elementul ocupă toată lățimea disponibilă
  • inline – elementul se aliniază cu textul
  • inline-block – se comportă ca inline, dar acceptă dimensiuni
  • none – ascunde elementul
Exemplu proprietatea display
display: block
display: inline
display: inline-block

Observați cum block ocupă o lățime fixă pe rând separat, inline este aliniat în linie cu textul, iar inline-block are dimensiuni dar este tot în linie.

2. Proprietatea position

Determină modul de poziționare a elementului:

  • static – poziționare normală în fluxul documentului (implicit)
  • relative – poziționare relativă față de poziția inițială
  • absolute – poziționare absolută față de cel mai apropiat părinte poziționat
  • fixed – poziționare fixă față de fereastra browserului
Exemplu proprietatea position
position: static (implicit)
position: relative; top: 10px; left: 20px;
position: absolute; top: 60px; left: 40px;
position: fixed; bottom: 20px; right: 20px;

Observați că relative se deplasează față de poziția inițială, absolute iese din flux și se poziționează absolut, iar fixed rămâne fixat în colțul ferestrei.

3. Proprietatea float

Permite „alinierea” elementelor la stânga sau dreapta, cu textul care curge în jur:

  • float: left; – elementul se lipește în partea stângă
  • float: right; – elementul se lipește în partea dreaptă
  • clear – împiedică elementele următoare să se înfășoare lângă elementele flotante
float: left
float: right

Acesta este un paragraf care înconjoară cele două blocuri flotante. Textul va curge în jurul elementelor flotante până când se aplică clear.
