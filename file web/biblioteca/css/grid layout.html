CSS - Pagina 7: Grid Layout

Ce este un Grid Container?

Un grid container este un element părinte care are display: grid; aplicat. El stabilește un context pentru copiii săi, numiți grid items, să fie aranjați conform liniilor și coloanelor definite.

display: grid;

Exemplu:

Item 1
Item 2
Item 3

Proprietatea grid-template-columns

Definește numărul și dimensiunea coloanelor din grid. Exemplu: grid-template-columns: repeat(3, 1fr); înseamnă 3 coloane egale, fiecare luând o fracțiune egală din spațiu.

grid-template-columns: repeat(3, 1fr);

Exemplu vizual cu 3 coloane:

Coloana 1
Coloana 2
Coloana 3

Proprietatea grid-template-rows

Definește numărul și înălțimea rândurilor. Exemplu: grid-template-rows: 100px 200px; creează 2 rânduri, primul înalt 100px, al doilea 200px.

grid-template-rows: 100px 200px;

Exemplu vizual cu 2 rânduri de înălțimi diferite:

Rând 1, Col 1
Rând 1, Col 2
Rând 2 (ocupă 2 coloane)

Span pe coloane cu grid-column: span n;

Permite unui grid item să ocupe mai multe coloane. Exemplu: grid-column: span 2; înseamnă că elementul ocupă două coloane consecutive.

grid-column: span 2;
