CSS - Pagina 3: Culori, Fonturi și Text

Proprietăți CSS pentru text

  • color – culoarea textului
  • background-color – culoarea fundalului
  • font-family – tipul de font
  • font-size – dimensiunea textului
  • text-align – alinierea textului (ex: left, right, center, justify)

Exemple

Acesta este un text roșu (proprietatea color: red).

Acest paragraf are un fundal verde deschis (proprietatea background-color: lightgreen).

Text cu font monospace (proprietatea font-family: monospace).

Text cu dimensiune mărită (proprietatea font-size: 24px).

Text centrat folosind text-align: center.

Formate de culoare

  • Nume: red, green, blue, etc.
  • HEX: #ff0000 (roșu)
  • RGB: rgb(255, 0, 0) (roșu)
