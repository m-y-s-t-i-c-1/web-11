Pseudo-clase și pseudo-elemente în CSS

1. Pseudo-clase

O pseudo-clasă se aplică unui element într-o anumită stare, cum ar fi când elementul este hover-uit, activ sau când este selectat.

  • :hover – se aplică unui element când utilizatorul trece cu mouse-ul peste el.
  • :active – se aplică unui element în momentul când este activat (de exemplu, când se face clic pe un buton).
  • :focus – se aplică unui element când acesta primește focus (de exemplu, când un câmp de formular este selectat).

2. Pseudo-elemente

Un pseudo-element este folosit pentru a aplica stiluri unor părți ale unui element, cum ar fi înainte sau după conținutul acestuia.

  • ::before – adaugă conținut înaintea unui element.
  • ::after – adaugă conținut după un element.

3. Pseudo-clasa :nth-child()

Permite aplicarea unui stil pe elemente pe baza poziției lor într-un părinte. De exemplu, :nth-child(odd) sau :nth-child(even) pentru a colora alternativ rândurile.
