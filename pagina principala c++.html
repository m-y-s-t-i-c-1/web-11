💻 Introducere în Programarea C++

🔹 Elemente de bază C++

C++ este un limbaj de programare de nivel înalt care oferă control complet asupra resurselor hardware.

  • Tipuri de date: int, float, char, bool
  • Variabile: Folosite pentru a stoca date
  • Instrucțiuni de bază: Atribuire, citire și scriere

🔹 Instrucțiuni simple

Structurile alternative sunt folosite pentru a lua decizii în funcție de condiții.

  • if: Verifică o condiție
  • if...else: Execută ramuri diferite
  • switch: Alternativă eficientă

🔹 Structuri repetitive

Structurile repetitive sunt folosite pentru a executa o secvență de cod de mai multe ori, în funcție de o condiție. Exemple:

  • while: Se execută cât timp o condiție este adevărată.
  • do...while: Se execută cel puțin o dată și apoi continuă dacă condiția este adevărată.
  • for: Utilizat pentru a repeta un bloc de cod de un anumit număr de ori.

🔹 Tablouri unidimensionale (Vectori)

Un tablou unidimensional este o colecție de elemente de același tip, accesibile printr-o singură indexare. Exemple de utilizare:

  • int v[10];: Definirea unui vector de 10 elemente de tip int.
  • Accesarea unui element: v[0], v[1], etc.

🔹 Tablouri bidimensionale (Matrici)

  • O matrice este un tablou de tablouri. Aceasta poate fi folosită pentru a reprezenta date în formă de tabel (linii și coloane).
  • int m[3][4]; // matrice cu 3 linii și 4 coloane

🔹 Funcții

Funcțiile sunt blocuri de cod care realizează o anumită sarcină și pot fi apelate de mai multe ori în program. Exemple:

  • Definirea unei funcții: int aduna(int a, int b)
  • Apelarea unei funcții: aduna(3, 4);
  • Recursivitatea: O funcție care se apelează pe ea însăși pentru a rezolva o problemă.

🔹 Structuri de date

C++ oferă diferite tipuri de structuri de date, printre care:

  • struct: Permite definirea de structuri personalizate, care pot conține variabile de diferite tipuri.
  • vector: Un tablou dinamic, care poate crește și micșora în funcție de necesități.
  • array: Un tablou cu dimensiune fixă, alocată la începutul programului.

🔹 Fișiere (citire/scriere din fișier)

În C++, putem interacționa cu fișiere folosind fluxuri de date, pentru a citi sau a scrie informații.

  • ifstream: Folosit pentru citirea din fișiere.
  • ofstream: Folosit pentru a scrie în fișiere.

🔹 Algoritmi clasici

Algoritmii clasici sunt folosiți pentru a rezolva probleme de sortare și căutare:

  • sort: Algoritmi de sortare, cum ar fi sortarea prin inserare, bubble sort, quicksort.
  • cautare binara: Folosită pentru a căuta eficient într-un tablou ordonat.

🔹 Algoritmi pe grafuri

Grafurile sunt structuri de date folosite pentru a reprezenta relațiile dintre obiecte. Algoritmi de bază:

  • DFS: Căutarea în adâncime.
  • BFS: Căutarea în lățime.
  • Reprezentarea grafului: Matrice de adiacență.

🔹 Alocare dinamică

Alocarea dinamică se face folosind operatorii new și delete, permițând alocarea de memorie în timpul execuției programului.

🔹 Clase și obiecte (Programare Orientată pe Obiecte)

Programarea orientată pe obiecte (OOP) permite organizarea codului în clase și obiecte, pentru a modela mai bine entitățile din lumea reală.

  • Clasele definesc atribute și metode.
  • Obiectele sunt instanțele unei clase.
