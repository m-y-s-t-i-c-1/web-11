Introducere în HTML

1. Ce este HTML?

HTML (HyperText Markup Language) este limbajul de marcare utilizat pentru a crea pagini web. Acesta definește structura și conținutul unei pagini web folosind diverse etichete.

Structura de bază a unei pagini HTML

  • <!DOCTYPE html> - Specifică tipul documentului (HTML5 în acest caz)
  • <html> - Eticheta principală care înconjoară întreaga pagină HTML
  • <head> - Contine metadatele paginii, cum ar fi titlul și stilurile
  • <body> - Conține conținutul vizibil al paginii web
2. Titluri și paragrafe
  • Etichete: <h1> până la <h6>, <p>
  • Cum se structurează textul în titluri și paragrafe
  • Exemple de ierarhie a titlurilor: <h1> pentru titlul principal și <h6> pentru titlurile de nivel inferior
3. Formatarea textului
  • Etichete: <strong>, <em>, <u>, <br>, <hr>
  • Cum adăugăm accent și stil textului folosind aceste etichete
4. Liste
  • Liste ordonate: <ol>, <li>
  • Liste neordonate: <ul>, <li>
  • Liste imbricate
5. Linkuri (Hyperlinkuri)
  • Eticheta <a> pentru crearea linkurilor
  • Atribute: href, target
  • Linkuri interne și externe
6. Imagini
  • Eticheta <img> pentru adăugarea imaginilor
  • Atribute: src, alt, width, height
  • Importanța atributului alt pentru accesibilitate
7. Tabele
  • Etichete: <table>, <tr>, <td>, <th>
  • Borduri și anteturi în tabele
  • Exemple de tabele simple
8. Formulare
  • Etichete: <form>, <input>, <textarea>, <button>, <select>, <option>
  • Atribute: action, method, name, value
9. Elemente div și span
  • Ce sunt containerele: <div> și <span>
  • Diferențele între ele
  • Cum se folosesc pentru structurare și stilizare
10. Comentarii și bune practici
  • Etichetă comentariu: <!-- comentariu -->
  • Importanța indentării și lizibilității codului
  • Cod HTML valid și semantic
